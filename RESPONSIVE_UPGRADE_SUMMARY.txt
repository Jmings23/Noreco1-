USAGE & INTEGRATION GUIDE
================================================================================

TO MAKE YOUR PAGES MOBILE RESPONSIVE:

1. ADD VIEWPORT META TAG (Required)
   Add to every PHP/HTML file in the <head> section:
   
   <meta name="viewport" content="width=device-width, initial-scale=1.0">

2. INCLUDE CSS IN CORRECT ORDER
   <link rel="stylesheet" href="../assets/css/theme.css">
   <link rel="stylesheet" href="../assets/css/dashboard.css">
   <link rel="stylesheet" href="../assets/css/responsive-utilities.css">

3. ADD HAMBURGER BUTTON (For pages with sidebar)
   <button class="sidebar-toggle">
       <span></span>
       <span></span>
       <span></span>
   </button>

4. INCLUDE JAVASCRIPT (Before closing </body>)
   <script src="../assets/js/mobile-menu.js"></script>

COMMON ISSUES & SOLUTIONS
================================================================================

Issue: Hamburger menu doesn't appear
Solution: Ensure mobile-menu.js is included and viewport meta tag is present

Issue: Sidebar doesn't collapse on mobile
Solution: Check that sidebar has correct classes and JavaScript is loaded

Issue: Tables are cut off on mobile
Solution: Tables should wrap in .table-responsive class or use overflow-x: auto

Issue: Text is too small on mobile
Solution: Check that font sizes are defined in mobile media queries

Issue: Images cause horizontal scroll
Solution: Ensure max-width: 100% is applied to all images

Issue: Forms aren't full width on mobile
Solution: Input fields should have width: 100% in mobile media queries

Issue: Tap targets are too small
Solution: Ensure buttons/links have at least 44x44px on mobile

Issue: iOS zoom on form input
Solution: Ensure inputs have font-size: 16px or larger
